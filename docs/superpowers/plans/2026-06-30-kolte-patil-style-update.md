# Implementation Plan: Premium UI/UX Redesign Audit

**Goal:** Elevate the website's look-and-feel from a standard corporate layout into a state-of-the-art, high-converting premium real estate landing page. Remove all boxy, basic table/list styles and replace them with bento grids, glassmorphism, dynamic image hover cards, custom capsule tab switches, and a floating hero form.

---

### Redesign Strategy (UI Audit Fixes)

1. **Header / Navbar**:
   - Thin elegant bottom border, premium sans-serif typography, hover underline transitions.

2. **Hero Section (Cinematic Split Layout)**:
   - Left side: Luxury tagline, bold text, bullet points.
   - Right side: Floating glassmorphic lead registration card with clean inputs, gold highlights, and shadow elevations.

3. **Highlights (Bento Grid Layout)**:
   - Create asymmetric highlights layout with distinct sizes, clean round indicators, and glowing backdrops.

4. **Overview Section**:
   - Large watermark stats (e.g. "21", "22") layered behind text for depth.

5. **Pricing Section**:
   - Borderless, deep shadow cards with clean configuration headers.

6. **Interactive Floor Plans**:
   - Rounded pill capsule nav.
   - Replace standard HTML tables with clean layout spec lists styled as rounded badge chips.

7. **Amenities Photo Cards**:
   - Renders each amenity as a full-cover picture card with a dark gradient bottom overlay and sliding text.

8. **Cinematic Video Placeholder**:
   - Render a glowing play icon pulsing in a dark blurred property view card.
